# UmirHack Blumenkohl

Система мониторинга и анализа информационного пространства. Собирает данные из Telegram, VK, RSS-лент и новостных сайтов, анализирует тональность, выявляет темы и ранжирует их по критичности.

## Команда

- **Михаил** — тимлид, модели анализа (BERTopic, классификация, эмбеддинги)
- **Андрей** — парсеры (Telegram, VK, RSS, сайты)
- **Алексей** — backend, база данных, бот

## Основные фичи

- Сбор данных из множества источников: Telegram-каналы и группы, VK, RSS-ленты, новостные сайты
- Фильтрация спама и токсичного контента
- Тематическое моделирование с помощью BERTopic
- Оценка срочности и критичности тем на основе ключевых слов и временного окна
- Ранжирование тем по индексу критичности
- Telegram-бот для взаимодействия с системой

## Требования

- Python 3.10+
- PostgreSQL
- Файл `.env` с переменными окружения:
  - `DB_USER`, `DB_PASSWORD`, `DB_NAME`, `DB_HOST`, `DB_PORT` — подключение к PostgreSQL
  - `BOT_TOKEN` — токен Telegram-бота
  - `VKAPI_TOKEN` — ключ для работы с ВК

## Установка

1. Установите зависимости:

```bash
pip install -r requirements.txt
```

2. Создайте файл `.env` и укажите необходимые переменные окружения.

## Запуск

**Шаг 1.** Скачайте и сохраните ML-модели локально:

```bash
python models/check.py
```

**Шаг 2.** Запустите бота:

```bash
python main.py
```

**Шаг 3.** Взаимодействуйте с ботом через Telegram. Доступные команды:

- `/start` — начать работу, показать меню с доступными категориями
- `/db` — пересоздать базу данных (удалить и создать заново)
- `/news` — запустить парсинг всех источников и сохранить данные в БД
- `/top` — проанализировать данные из БД и показать топ тем по индексу критичности

## Архитектура

```
bot/            — Telegram-бот, обработчики команд, клавиатуры
models/         — ML-модели, анализ данных
parsers/        — Парсеры Telegram, VK, RSS, сайтов
```
